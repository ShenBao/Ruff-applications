# fire extinguisher

## 介绍

我们看过许多火灾的新闻报道，火灾造成了巨大的生命财产损失，因此，火灾的监测及防护成了重中之重

这是一个火灾监测及防护应用。当火焰传感器检测到火焰信号时，保安室的LCD显示屏提示有火灾，同时控制灭火器的继电器
打开，开始灭火
