# Vibration to rgb led

## 介绍

看过小孩子脚上那Bling Bling的鞋吗？你肯定见过的，可爱的小孩子脚每
踩一下地，鞋子就亮一下

这个项目的灵感就来源于此，每当震动传感器接收到震动信号时，
LED灯就会交替颜色的发亮

## 硬件要求

1.[RGB LED Module(Model:KY-016)](https://rap.ruff.io/devices/KY-016)
