# weather-station

## 介绍

想要办公室或家维持在一定的舒适的温度和湿度范围内吗？这个温度湿度监测系统可以帮你

这个案例利用温湿度传感器从环境中换取温度和湿度数据，如果温度和湿度不在人体舒适的
范围内，则通过相应的继电器控制空调或加湿器的开关，从而达到控制环境温湿度的目的。

这个项目中，我们默认人体舒适的温度范围为20℃-25℃，舒适的湿度范围为40%RH-70%RH(相对湿度)
